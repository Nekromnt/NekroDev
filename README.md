GitHub Pages publishing:

1. Create a new GitHub repository.
2. Upload everything from this folder.
3. Open repository settings.
4. Go to Pages.
5. Choose deploy from branch, select `main`, folder `/root`, then save.

Update an already published GitHub Pages site:

1. Open your GitHub repository.
2. Click `Add file`.
3. Click `Upload files`.
4. Drag all files and folders from this folder into GitHub.
5. If GitHub asks about replacing files, keep the new versions.
6. Click `Commit changes`.
7. Wait until the yellow/green Pages build indicator finishes.
8. Open the site in an incognito/private window or hard refresh the page.

Photo:
